# EAVES — Logo

| File | Use |
|------|-----|
| `eaves_logo_light.png` | Horizontal lockup for **light** backgrounds — 2381×567, 600 DPI |
| `eaves_logo_dark.png` | Horizontal lockup for **dark** backgrounds (#0F2E52) — 2381×567, 600 DPI |
| `eaves_logo_light.svg` | Vector master (light) — infinitely scalable, text as outlines, no font needed |
| `eaves_logo_dark.svg` | Vector master (dark) — infinitely scalable, text as outlines, no font needed |
| `eaves_appicon_light.png` | App / favicon tile, light |
| `eaves_appicon_dark.png` | App / favicon tile, dark |

For print or any large use, prefer the **SVG** — it stays sharp at any size.

### Palette
- Wordmark navy `#14365C` · triangle-A cyan `#1FA9E6` (on dark `#2BB4EF`)
- Dam / cliff `#3C4654` · crest `#2A323D` · dark background `#0F2E52`
- Water steps: `#CFEAF9` `#8FD0F2` `#3AA0E0` `#1F7FC4` `#15518F`

### Type
Wordmark set in **Space Grotesk Bold** — "E" + cyan triangle "A" + "VES".

### Don't
Recolor, stretch, rotate, re-add a tagline, or move the wordmark off its baseline.
